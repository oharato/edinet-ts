
## セットアップ (初回のみ)

本ライブラリの時系列分析機能を使用するには、EDINETの過去の書類リスト（メタデータ）をローカルデータベースに構築する必要があります。
API制限（1秒1リクエスト）を遵守するため、全データ（過去5年分）の取得には約20分程度かかります。

1. プロジェクトルートに `.env` ファイルを作成し、APIキーを設定してください。
   ```
   EDINET_API_KEY=your_api_key_here
   ```

2. 以下のコマンドを実行してデータベースを構築します。
   ```bash
   npx tsx scripts/seed.ts
   ```
